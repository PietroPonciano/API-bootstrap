## Requisitos

- Node.js 18+

## Instalação

```bash
npm install
```

## Configuração

Copie o arquivo `.env.example` para `.env` e ajuste as variáveis conforme necessário:

```bash
cp .env.example .env
```

## Execução

Modo produção:

```bash
npm start
```

Modo desenvolvimento (com reinício automático):

```bash
npm run dev
```

A API estará disponível em `http://localhost:3000`.
